## 二、ISO 9223 大气腐蚀性分级（碳钢基准）

| 等级 | 描述 | 第一年质量损失 (g/m²) | 第一年厚度损失 (μm) | 长期腐蚀速率 (μm/年) |
|:---:|------|:-------------------:|:------------------:|:-------------------:|
| **C1** | 极低（清洁室内） | ≤10 | ≤1.3 | ≤0.1 |
| **C2** | 低（乡村） | 10–200 | 1.3–25 | 0.1–1.5 |
| **C3** | **中（城市）** | **200–400** | **25–50** | **1.5–6** |
| **C4** | 高（工业/沿海） | 400–650 | 50–80 | 6–20 |
| **C5** | 极高（重工业/近海） | 650–1500 | 80–200 | 20–90 |
| **CX** | 极端（离岸/盐雾） | 1500–5500 | 200–700 | 90–250 |

> **深田社区（厦门城市环境）应归为 C3–C4**（城市 + 沿海中等盐度）

## 四、关键腐蚀规律

### 4.1 腐蚀速率随时间递减

碳钢在大气中首年腐蚀速率最高，之后因锈层覆盖而递减：

| 时间 | 平均速率 (μm/年) | 相对首年 |
|------|:---------------:|:------:|
| 第 1 年 | 40 | 100% |
| 第 2 年 | 28 | 70% |
| 第 3–5 年 | 18 | 45% |
| 第 5–10 年 | 12 | 30% |
| 10 年以上 | 5–8 | ~15% |

**拟合公式（幂函数）**：
```
腐蚀深度 d = A × t^n
```
其中 n ≈ 0.5–0.6（因环境和钢种而异），A 为首年腐蚀深度。

### 4.2 湿度阈值

- 相对湿度 **< 60%**：腐蚀极慢（表面水膜不连续）
- 相对湿度 **60–80%**：腐蚀加速（临界湿度）
- 相对湿度 **> 80%**：腐蚀显著（水膜连续，氧扩散快）
- **厦门年均 RH ≈ 77%** → 常年处于加速区间

### 4.3 SO₂ 和 Cl⁻ 的影响

| 污染物 | 来源 | 对腐蚀速率的影响 |
|--------|------|----------------|
| SO₂ | 工业排放、汽车尾气 | C3 环境主导因子，浓度升高 1 μg/m³ → 腐蚀 +2–5% |
| Cl⁻ | 海洋盐雾 | 距海岸 < 1km 时显著，距海岸 > 10km 时影响渐小 |

## 五、实际意义：公共设施的碳钢组件

### 5.1 不防护碳钢的典型寿命

假设 2mm 厚碳钢板（城市环境，C3，首年 40μm → 长期 8μm/年均值）：

| 功能失效模式 | 估计时间 |
|------------|:------:|
| 表面锈斑出现（美观失效） | 3–6 个月 |
| 漆膜鼓泡剥落（涂层失效） | 2–5 年 |
| 截面减薄 10%（结构受损） | ~15 年 |
| 穿孔（厚度完全腐蚀） | ~50–60 年（但结构早已失效） |

**对社区设施而言：美观失效和涂层失效是最先出现的，通常 2–5 年内需要重新涂装。**

### 5.2 有防护涂层的情况

标准户外钢结构防护（ISO 12944-5 推荐）：
- 热浸镀锌 ≥ 85μm → 城市环境可用 20–40 年（无需维护）
- 环氧富锌底漆 + 聚氨酯面漆 → 10–15 年（需重涂）

## 六、对贝叶斯模型的推荐参数

### 6.1 材质差异（相对于碳钢基准）

```python
METAL_CORROSION_RATE = {
    # 材质类型 → 相对腐蚀速率（以碳钢 = 1.0 为基准）
    # C3 城市环境基准速率：40 μm/年（首年）
    "碳钢(未涂装)":  1.00,   # 基准
    "碳钢(涂装完好)": 0.05,  # 涂层阻挡 ~95%
    "碳钢(涂装老化)": 0.30,  # 涂层部分失效
    "镀锌钢":        0.08,   # 锌牺牲保护
    "不锈钢304":     0.005,  # 几乎不腐蚀
    "铸铁":          0.85,   # 略优于碳钢（石墨化腐蚀）
}
```

### 6.2 环境加速因子（用于 exposure CPT）

```python
METAL_ENV_FACTOR = {
    # (sun_shade, water_log) → 相对基准的加速因子
    # 基准条件：有遮 + 低积水 → 因子 1.0
    ("暴晒", "低"): 0.8,   # 干燥 → 腐蚀略慢
    ("暴晒", "中"): 1.2,   # UV + 间歇湿润
    ("暴晒", "高"): 1.5,   # UV + 频繁湿润
    ("有遮", "低"): 1.0,   # 基准条件
    ("有遮", "中"): 1.5,   # 潮湿主导（无 UV 干燥）
    ("有遮", "高"): 2.0,   # 长期潮湿 → 腐蚀最快
}
```

> 注意：与木材不同，金属在**有遮+高积水**条件下退化最快（潮湿无阳光干燥），暴晒反而有干燥效应。
