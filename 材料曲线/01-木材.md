## 二、表面侵蚀速率（核心数据）

### 2.1 北美松木类（USDA 长期户外暴露，垂直 90° 朝南）

| 树种 | 早材侵蚀 (μm/年) | 晚材侵蚀 (μm/年) | 平均 (mm/年) |
|------|-----------------|-----------------|-------------|
| Loblolly pine | 32–34 | 18–20 | ~0.025 |
| Southern pine | 46–62 | 16–20 | ~0.035 |
| Ponderosa pine | ~40 | ~33 | ~0.037 |
| Lodgepole pine | ~40 | ~33 | ~0.037 |
| Douglas-fir | 37–42 | 10–20 | ~0.028 |
| Western redcedar | 96–131 | 26–35 | ~0.07–0.08 |

### 2.2 红橡木（不同暴露角度）

| 暴露角度 | 早材 (μm/年) | 晚材 (μm/年) |
|---------|-------------|-------------|
| 90°（垂直） | 35 | 30 |
| 45° | 67 | 80 |
| 0°（水平） | 114 | 105 |

**结论：45° 暴露侵蚀速率约为垂直的 2 倍，水平暴露约为垂直的 3 倍。**

### 2.3 长期平均值

| 木材类型 | 典型侵蚀速率 |
|---------|-------------|
| 多数软木（松木类） | ~6 mm/百年 = **0.06 mm/年** |
| 多数硬木（橡木类） | ~3 mm/百年 = **0.03 mm/年** |
| 松木早材（水平暴露，恶劣条件） | 可达 **0.12 mm/年** |
| 橡木晚材（垂直暴露，温和条件） | 低至 **0.027 mm/年** |

## 三、水的倍增效应（关键）

### 3.1 加速老化中的粗糙度变化（红松，720h）

| 条件 | 粗糙度增幅 |
|------|----------|
| 30°C + UVA340，无雨 | +18.32% |
| 60°C + UVA340，无雨 | +43.33% |
| 60°C + UVA340，**+ 淋雨** | **+193.36%** |

→ 水分使退化加速约 **4.5–10 倍**。

### 3.2 水分作用机制

1. 细胞壁反复湿胀干缩 → 微裂纹产生
2. 雨水淋洗带走光降解产物 → 暴露新鲜表面继续降解
3. 湿润表面延长了紫外线作用的有效时间
4. 加速老化中初始含水率 25% 的试件加速系数：**5.1–11.4** 倍

## 四、化学退化机制

### 4.1 木质素降解（太阳光 UV 的主要靶标）

- 木质素最先且最快被光降解（强烈吸收 280–400nm UV）
- 无雨：降解呈深度梯度（由表及里）
- 有雨：从外表面和细胞腔两侧同时进行
- 日本山毛榉 24h 加速老化不同组织的木质素降解率：
  - 导管分子：**73.3%**
  - 轴向薄壁细胞：**57.2%**
  - 木纤维：**45.3%**

### 4.2 羰基生成

- 羰基含量与总色差 ΔE* 呈正相关
- 干模式：羰基堆积（氧化产物）
- 湿模式：羰基被淋洗流失，颜色演化路径完全不同于干模式

## 五、对贝叶斯模型的建议参数

### 推荐的退化速率映射（用于 engine.py 的 exposure CPT 评分）

```python
# 基于 USDA FPL 自然风化 + 中国红松加速老化文献
WOOD_DEGRAD_RATE = {
    # (sun_shade, water_log) → 年退化速率 (mm/年)
    # 基准：松木垂直暴露 0.04 mm/年
    ("暴晒", "低"): 0.04,   # 基准：UV辐射主导
    ("暴晒", "中"): 0.07,   # ~1.75× 加速（水协同）
    ("暴晒", "高"): 0.12,   # ~3× 加速（水+UV 强协同）
    ("有遮", "低"): 0.015,  # ~0.37×（无UV → 主要靠湿度循环）
    ("有遮", "中"): 0.03,   # ~0.75×
    ("有遮", "高"): 0.06,   # ~1.5×（水主导，无UV加成）
}
```

### 退化速率 → exposure CPT 评分映射

当前 `_score_cpt` 使用 0–8 分值。建议将退化速率映射到该分值：

```python
rate = WOOD_DEGRAD_RATE[(sun_shade, water_log)]
if rate < 0.02:    score = 0–1  # 低暴露
elif rate < 0.05:  score = 2–3  # 中暴露
elif rate < 0.08:  score = 4–6  # 中高暴露
else:              score = 7–8  # 高暴露
```
