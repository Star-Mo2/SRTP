## 二、户外寿命关键数据

### 2.1 拉伸强度随时间退化

| 暴露时间 | 拉伸强度损失 | 条件 |
|---------|:----------:|------|
| 28 天 | ~30% | 盐水+UV（最恶劣） |
| 3 个月 | ~8% | 大气暴露 |
| 8 个月 | ~10% | 大气暴露 |
| 4 年 | 断裂应变 -80%（UTS 未显著下降） | 韩国户外 |
| 35 年 | UTS 损失 ~60% | 海洋大气（希腊） |

> ⚠️ 注意：**断裂应变**（韧性）比 **拉伸强度**退化得更快更早。HDPE 制品往往在强度显著下降之前就已变脆开裂。

### 2.2 有无稳定剂的差异（1152h 加速 UV）

| 性能指标 | 原始 HDPE | 未稳定化 UV 后 | 加 2% 炭黑 UV 后 |
|---------|:--------:|:------------:|:--------------:|
| 拉伸强度 (MPa) | 25.3 | 14.6 (−42%) | 22.2 (−4.2%) |
| 断裂伸长率 (%) | 65.4 | 12.5 (−81%) | 39.3 (−29%) |
| 承载能力 (N) | 955 | 638 (−33%) | 992 (+4%) |
| 杨氏模量 (MPa) | 689 | 645 (−6%) | 951 (+38%) |
| 硬度 (Shore D) | 61 | 66 (+8%) | 62 (+1.6%) |

→ **炭黑是 HDPE 最简单有效的 UV 稳定剂，2% 添加量即可阻止大部分退化。**

### 2.3 自然暴露 → 脆弱化的时间线

| HDPE 类型 | 显著脆化时间 | 说明 |
|----------|:----------:|------|
| 完全无稳定剂 | **60–90 天** | 佛罗里达户外暴露 |
| 无稳定剂（温和气候） | **300h 加速 UV** | 韧性丧失 98% |
| HDPE + HALS/UVA | 900–1300h 加速 UV | 10–14× 优于无稳定剂 |
| HDPE + 2% 炭黑 | **10 年以上** | 户外家具标准配方 |
| 商业级户外 HDPE 制品 | **5–15 年** | 取决于稳定剂配方和气候 |

## 三、UV 剂量-损伤关系

### 3.1 韩国 4 年研究的关键发现

- 总 UV 剂量 (315–400nm)：**1,020 MJ/m²**（4 年累计）
- 断裂应变从 897% → 175%（保留率 ~20%）
- **UV 剂量仅 355 MJ/m² 即导致断裂应变减半**
- 拉伸模量和屈服强度：**无显著变化**（退化局限于表面 100–400μm）

### 3.2 加速老化 → 自然老化的换算

| 加速条件 | ≈ 自然暴露 |
|---------|:---------:|
| 1.5 个月加速 UV | 6 个月大气暴露 |
| 240.5 小时 QUV | 337.5 天自然 |
| 3–4 个月加速 UV | 3–4 年海面暴露 |

## 四、降解机制

### 4.1 Norrish I/II 光化学反应链

```
UV 光子吸收 → C–H / C–C 键断裂 → 自由基生成
→ 与 O₂ 反应 → 过氧自由基
→ 夺氢 → 氢过氧化物 (ROOH)
→ 光解/热解 → 烷氧自由基 + 羟基自由基
→ 链式反应 → 羰基 (C=O) + 乙烯基 (C=C) 积累
```

### 4.2 表面脆化层

- 厚度：**100–400 μm**
- 此层内分子链断裂、交联密度增加 → 脆性
- 内部材料可能仍然完好（"皮脆心嫩"）
- 表面微裂纹从脆化层起始，在应力下向内部扩展

### 4.3 环境协同因素

| 因素 | 效应 |
|------|------|
| 高温（>40°C） | 加速自由基反应，热氧化与光氧化叠加 |
| 水分 | 缓慢水解（HDPE 耐水性好，影响相对小） |
| 机械应力 | 应力开裂加速（环境应力开裂 ESC） |
| 污染物 | 酸雨/臭氧攻击表面 |

## 五、PP（聚丙烯）对比

很多社区户外设施用 PP 而非 HDPE（如注塑座椅板条）：

| 特性 | HDPE | PP |
|------|------|-----|
| UV 稳定性（无稳定剂） | 差 | **极差** |
| 户外寿命（无稳定剂） | 60–90 天脆化 | **30–60 天脆化** |
| UV 稳定性（+ 炭黑/HALS） | 好 | 良好 |
| 低温脆性 | 好 (–100°C) | 差 (0°C 以下变脆) |

> **PP 比 HDPE 更容易 UV 降解，且低温韧性差，在户外设施中通常需要更重的稳定剂配方。**

## 六、对贝叶斯模型的推荐参数

### 6.1 材质基准退化速率

```python
PLASTIC_DEGRAD_RATE = {
    # 材质类型 → 年退化速率（以"有效寿命消耗比例"表示）
    # 基准：商业级户外 HDPE（含 UV 稳定剂）在温和条件下 ~12 年寿命
    # → 年消耗率 = 1/12 ≈ 0.083

    "HDPE(含UV稳定剂)": 0.08,   # ~12 年寿命，基准
    "HDPE(无稳定剂)":   0.50,   # ~2 年寿命
    "HDPE(含炭黑)":     0.05,   # ~20 年寿命
    "PP(含UV稳定剂)":   0.12,   # ~8 年寿命
    "PP(无稳定剂)":     0.80,   # ~1.25 年寿命
    "PVC(硬质户外)":    0.06,   # ~16 年寿命
}
```

### 6.2 环境加速因子

```python
PLASTIC_ENV_FACTOR = {
    # (sun_shade, water_log) → 相对基准的加速因子
    # 塑料退化主要由 UV 驱动，水分影响相对较小

    ("暴晒", "低"): 1.5,   # UV 主导 + 干燥 = 热氧化叠加光氧化
    ("暴晒", "中"): 1.3,   # UV 主导 + 间歇湿 → 水分带走碎片
    ("暴晒", "高"): 1.2,   # UV 主导 + 水冷 → 表面温度降低，略减缓
    ("有遮", "低"): 0.3,   # 无 UV → 仅热氧化，退化极慢
    ("有遮", "中"): 0.35,  # 无 UV + 潮湿
    ("有遮", "高"): 0.4,   # 无 UV + 长期潮湿（可能促进水解）
}
```

> 与木材和金属不同：**塑料最怕暴晒**，遮阴对塑料是保护。暴晒+低积水（干燥+热）反而最差（热氧化叠加），暴晒+高积水因水冷降低表面温度而略有缓和。

| 环境条件 | 木材 | 碳钢 | HDPE |
