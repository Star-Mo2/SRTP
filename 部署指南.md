# 部署到 PythonAnywhere（免费）

## 第一步：注册 PythonAnywhere

1. 打开 https://www.pythonanywhere.com
2. 点击 "Start running Python online" → 创建免费账号
3. 免费版：1 个 Web 应用、有限流量（个人/小团队够用）

## 第二步：上传代码

**方式 A（推荐）：通过 GitHub**
```bash
# 在你电脑上，进入项目目录
cd "E:\111AAAFZUSpecialized courses\11AA大三下\SRTP\论文"

# 初始化 Git（如果还没做）
git init
git add .
git commit -m "initial commit"

# 推送到 GitHub
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

然后在 PythonAnywhere 的 Bash 控制台：
```bash
git clone https://github.com/你的用户名/你的仓库名.git
```

**方式 B：直接上传**
- PythonAnywhere → Files 标签 → 上传所有 .py / .html / .css / .js 文件

## 第三步：安装依赖

在 PythonAnywhere 的 Bash 控制台：
```bash
pip install --user flask pgmpy pandas numpy requests
```

## 第四步：配置 Web 应用

1. PythonAnywhere → Web 标签 → "Add a new web app"
2. 选择 Flask + Python 3.11
3. 编辑 WSGI 配置文件，把 flask_app 指向你的 app.py：

```python
import sys
path = '/home/你的用户名/你的项目目录'
if path not in sys.path:
    sys.path.append(path)

from app import app as application
```

## 第五步：创建 config.json

在 PythonAnywhere 的 Files 中，手动创建 config.json（因为 .gitignore 排除了它）：
```json
{
    "llm": {
        "api_key": "你的API Key",
        "endpoint": "https://api.deepseek.com/v1/chat/completions",
        "model": "deepseek-chat"
    },
    ...
}
```

## 第六步：访问

点击 Web 标签中的链接 → `https://你的用户名.pythonanywhere.com`

---

## 注意事项

- 免费版不能使用自定义域名
- 免费版每 3 个月需登录一次（会发邮件提醒）
- config.json 不要上传到 GitHub（已加入 .gitignore）
- 如果 5 分钟内无访问，应用会休眠（下次访问自动唤醒，首刷稍慢）
